# `vite-plugin-wasm`: Vanilla vite app w/ _npm crate import_

# Quickstart

```bash
git clone https://github.com/michaelwooley/vite-plugin-wasm

cd vite-plugin-wasm-pack

# Build the plugin itself
npm i
npm run build

# Set up the Example
cd ex/npm-crate
npm i

# READY TO GO IN JS:
npm run dev

# For prod:
npm run build
npm run preview
```
